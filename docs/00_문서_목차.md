# CAD Studio 개발 문서 목차

이 폴더는 AI 개발 도구가 CAD Studio의 제품 방향과 구현 규칙을 읽고 따르기 위한 제어 문서 모음이다.

## 읽는 순서

1. 이 문서
2. `01_제품_기획_정책.md`
3. 현재 작업과 관련된 정책 문서
4. `06_AI_개발_작업_명령서.md`
5. `05_사용자_승인_정책.md`

## 문서 목록

| 문서 | 용도 |
|---|---|
| `01_제품_기획_정책.md` | 제품 목표, MVP, 우선순위 |

## 공통 원칙

- 모든 도면 변경은 command를 통해 수행한다.
- UI는 도면 데이터의 소유자가 아니다.
- 계산 결과는 deterministic해야 한다.
- 파일 손실 가능성이 있는 작업은 먼저 사용자 승인을 받는다.
- 구현 후 테스트와 검증 결과를 반드시 기록한다.
