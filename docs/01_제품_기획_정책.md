# 제품 기획 정책

## 제품 정의

CAD Studio는 개인 설계자가 건축 및 기계 2D 도면을 안전하고 반복 가능하게 제작하는 로컬 중심 Rust 프로그램이다.

핵심 목표는 동시 접속이 아니라 한 사용자가 장시간 많은 도면을 안정적으로 생산하는 것이다.

## 우선 가치

1. 도면 데이터 손실 방지
2. 자동 저장과 복구
3. 반복 작업 및 대량 출력
4. 공차 계산의 추적 가능성
5. 같은 입력에 같은 결과를 내는 결정성
6. GUI와 CLI에서 동일한 엔진 재사용

## 비목표

- 대규모 동시 접속
- 실시간 협업
- 클라우드 전용 저장
- 3D 모델링
- 완전한 DWG 호환
- 모든 국제 표준의 완전한 GD&T 지원

## MVP

반드시 포함한다.

- 프로젝트 생성, 저장, 불러오기
- 자동 저장, 백업, 복구
- 선, 폴리라인, 사각형, 원, 호, 텍스트
- 선택, 이동, 삭제, pan, zoom, grid
- 레이어와 기본 스타일
- 선형 치수와 정렬 치수
- 대칭 공차, 상하 공차, 한계 공차
- 공차 최소값/최대값 계산과 계산 trace
- SVG 출력
- CLI 검증 명령

후순위로 둔다.

- SQLite 기반 저장
- PDF와 DXF 출력
- 블록과 템플릿
- 끼워맞춤 표
- 공차 누적 계산
- GD&T 기호

## 기능 판단

새 기능은 다음을 모두 검토한다.

- 개인 도면 생산성을 높이는가?
- 데이터 안전성을 해치지 않는가?
- command로 표현 가능한가?
- GUI와 CLI에서 재사용 가능한가?
- 테스트 가능한가?
- 기존 모듈 경계를 지키는가?
